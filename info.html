CodeDriven

CodeDriven, is a site I built that has blogs on coding concepts to simplify things for learning as well as code references. CodeDriven also has a community page with a registrations system and login built from scratch with PHP and mysqli where if you are logged in you can leave a post, delete and edit it as well. I made sure to use form validations with javascript and php prepared statements to prevent hackers from hacking into the database. --Visit Site--

ShoppingCart

Here is a project I made where when you choose a product it is saved inside the local storage. Local storage holds values such as quantity, products, and the total which makes it so that when the page reloads the values are still there. There is also a cart page which displays the items and you can also adjust the items and quantities before checkout. --Visit Site--
